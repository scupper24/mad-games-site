## Визуальные референсы

Скриншоты сайта INK GAMES (inkgames.com) — были предоставлены как изображения в чате.

Ключевые наблюдения из референсов:
- **Фон:** чисто чёрный (#000000 или очень тёмный)
- **Типографика:** жирные uppercase заголовки, шрифт с характером (близкий к condensed bold)
- **Hero-секция:** крупный текст по центру, 3D-объекты игры "летят" вокруг
- **Карточки игр:** квадратные/прямоугольные, тёмный overlay снизу с текстом
- **Раздел "карусель" игры:** центральная крупная карточка + боковые меньшего размера
- **Новостная лента:** минималистичная, дата / категория / заголовок в строку
- **Футер:** светлый (#f0f0f0), 3 колонки (навигация / соцсети / контакты) + адрес
- **Фиолетовый экран:** яркий фон (#aa44ff) с плавающими карточками игр под углом
- **Кнопки:** маленькие, bordered, uppercase с иконкой-стрелкой ▾
